# Cybersecurity GRC Risk Assessment — NIST SP 800-30

## Overview
Developed a comprehensive risk register for a fictional mid-sized fintech 
enterprise (FinTech Solutions, 500 employees) using the NIST SP 800-30 framework 
to identify, analyze, and prioritize cyber threats.

## What's Included
- **Risk Register** — 24 unique threats across 6 categories (Adversarial, Insider 
  Threat, Accidental, Environmental, Compliance, Operational) with inherent and 
  residual risk scoring
- **Risk Heat Map** — 5×5 likelihood × impact matrix with all threats plotted and 
  color-coded by severity
- **Executive Dashboard** — KPI summary, category breakdown, and key findings for 
  stakeholder reporting

## Key Outcomes
- Identified and scored 24 cyber threats including ransomware, SQL injection, 
  insider threats, supply-chain attacks, and physical security risks
- Mapped technical vulnerabilities to quantified business impacts (e.g., $50K/hr 
  portal downtime, $2M ransomware exposure)
- Documented mitigation strategies reducing estimated portfolio residual risk by ~73%

## Threat Categories Covered
| Category | Threats |
|---|---|
| Adversarial | Phishing, Ransomware, SQL Injection, DDoS, MitM, Brute Force, Supply Chain, Cloud Misconfiguration, API Key Leakage, Zero-Day |
| Insider Threat | Data Exfiltration, Accidental Disclosure, Privilege Abuse, Shadow IT |
| Accidental | DB Deletion, Firewall Misconfiguration, Lost Laptop |
| Environmental | Power Outage, Flood, Unauthorized Physical Access |
| Compliance | Breach Notification Failure, Unencrypted PII |
| Operational | Third-Party Breach, Expired TLS Certificate |

## Framework
NIST SP 800-30 Rev. 1 — Guide for Conducting Risk Assessments
